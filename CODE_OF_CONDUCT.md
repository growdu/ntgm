# 贡献者公约

## 我们的承诺

为了营造开放友好的社区环境，我们承诺无论年龄、体型、残疾、种族、性别特
征、性别认同、性别表达、经验水平、教育程度、社会经济地位、国籍、个人外
貌、种族、宗教或性别认同与性取向如何，都欢迎每位参与者。

## 我们的标准

有助于创造积极环境的行为包括：

- 表现出对他人的善意和同理心
- 尊重不同的意见、观点和经验
- 优雅地接受建设性反馈
- 专注于对社区最有利的事情
- 对其他社区成员保持耐心

不当行为包括：

- 使用性别化的语言或图像，以及不受欢迎的性关注或性骚扰
- 恶意评论、人身攻击或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的隐私信息（如物理地址或电子地址）
- 在专业环境中可能被视为不当的其他行为

## 执行责任

社区维护者负责阐明和执行可接受行为的标准，对于任何不可接受行为的实例，
将采取适当和公平的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，包括但不限于 GitHub 仓库、邮件列表、聊天
群组、社交媒体以及任何社区代表出席的线上线下活动。

## 执行

如遇滥用、骚扰或其他不当行为，请联系维护者邮箱 dev@ntgm.app。所有投诉
将被审查和调查，并按情况采取必要措施。

## 归属

本行为准则改编自
[Contributor Covenant](https://www.contributor-covenant.org)，版本 3.0，
发布于 https://www.contributor-covenant.org/version/3/0/code_of_conduct.html

社区影响指南中详述了对违反本行为准则的回应措施：
https://www.contributor-covenant.org/enforcement-guidelines
